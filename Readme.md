## How to run our code

1. Firs, run the data preparer
    - `python3 ./data_preparer.py`
2. Next, run the BERT model
    - `python3 ./bert.py`
3. Last, run the statistical analysis
    - `python3 ./statistical_analysis.py`

Note that Python is commonly installed as `py` on Windows
